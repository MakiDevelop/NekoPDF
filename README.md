NekoPDF 是一款專為 macOS／iOS 使用者設計的輕量 PDF 工具，
讓你用最少的步驟，將多張圖片整理、排序並輸出為單一 PDF 檔案。

沒有複雜設定，沒有多餘功能，
只專注把「圖片轉成 PDF」這件事，做到順、做到快、做到不干擾思考。

核心功能

• 從相簿或檔案選取多張圖片
• 以 Grid 方式預覽所有頁面
• 直覺式拖曳排序、刪除頁面
• 每張圖片自動轉為獨立 PDF 頁面
• 一鍵匯出標準 PDF，方便分享、列印與保存

NekoPDF 特別適合以下情境：
掃描文件、收據整理、簡報草稿、作業繳交、教學講義、快速歸檔。

為什麼選擇 NekoPDF

多數 PDF 工具試圖做「所有事」，
NekoPDF 只做一件事，但把流程設計到最順。

• 不需要登入
• 不上傳雲端
• 所有處理都在裝置本地完成
• 對隱私友善，沒有多餘權限

你不需要學習一個新系統，
只需要拖曳、排序、輸出。

設計理念

NekoPDF 的設計理念很簡單：
工具應該消失，結果才重要。

如果你需要的是一個可靠、安靜、隨時可用的 PDF 整理工具，
NekoPDF 會是那隻在你桌角默默幫忙的貓。
